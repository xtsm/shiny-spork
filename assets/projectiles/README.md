Формат снарядов
======

Снаяряд с идентификатором S представлен папкой S
В папке S содержатся файлы:

 * sprite.png - текстура,
 * config.txt - конфиг.

## S.png
Содержит спрайт.

## S.txt 
Имеет следующий формат:

    speed
    splash
    range

где `speed` - скорость полёта.
`splash` - урон от осколков.
`range` - дальность разлёта осколков.
